# 🚀 Automação de Testes com Cypress

Este repositório documenta minha jornada de aprendizado em **automação de testes utilizando Cypress**, saindo do nível de QA Manual para automação.

Aqui você encontrará desde a configuração inicial do projeto até a criação de testes automatizados, seguindo boas práticas utilizadas no mercado.

---

## 🎯 Objetivo

* Evoluir para além do QA Manual
* Aprender Cypress na prática
* Construir um portfólio sólido na área de testes
* Documentar todo o processo de aprendizado

---

## 🧰 Tecnologias utilizadas

* Node.js
* NPM
* Cypress
* JavaScript
